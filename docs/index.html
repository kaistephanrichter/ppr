Features

Everything you need,
nothing you don't.

PPR connects directly to your paperless-ngx server — no cloud middleman, no subscription.

Document Capture

Scan with the camera, import from Photos, or pick any file. Multi-page scanning with automatic cropping.

Smart Upload

Set tags, document type, correspondent, and title before uploading. Metadata is synced live from your server.

Document Management

Browse, search, and filter your entire archive. View PDFs inline with full-screen support.

Dark Mode & Localization

Fully translated in 7 languages (DE, EN, FR, ES, IT, NL, PL) with custom accent colors for light and dark appearance.

Server Status

Monitor your paperless-ngx instance health — task queue, index status, storage usage — directly in the app.

Privacy First

All data stays on your server. Credentials are stored securely in the iOS Keychain. Zero telemetry.

Requirements

What you need

PPR connects to your existing paperless-ngx setup. No new infrastructure required.

iOS

iOS 18.0 or later

Server

paperless-ngx instance with REST API enabled

Network

Device must reach your server (local network or VPN)

API Token

Found in paperless-ngx under Settings → Profile

Setup

Get started in minutes

PPR walks you through setup on first launch. Just have your server URL and API token ready.

1

Find your API token

In your paperless-ngx web interface, go to Settings → Profile → API Token and copy it.

2

Open PPR and follow onboarding

On first launch, PPR shows the onboarding screen. Enter your server URL (e.g. http://192.168.1.100:8000) and paste your API token.

3

Start scanning

PPR connects to your server and loads your tags, document types, and correspondents. You're ready to scan.
